# Security Policy

## Reporting a vulnerability

Email **support@sonovault.now** with the details. Do not open a public issue for security problems.

We aim to acknowledge reports within 2 business days.

## Scope

This repository contains a client library. Vulnerabilities in the SonoVault API itself can be reported to the same address.

## API keys

Never commit an API key. The library reads keys at runtime only. If you exposed a key, rotate it in your dashboard at https://sonovault.now.
